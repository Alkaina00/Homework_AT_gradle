3. Каким образом JAR-файлы этих библиотек попали к нам в проект?
Ответ: Подгрузились с помощью гредла. То есть мы вписали их в зависимостях (implementation) в файле build.gradle 
и gradle скачал JAR-файлы из репозитория Maven и добавил их в проект
